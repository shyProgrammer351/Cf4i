문제

구슬을 3개 컵 중에서 어디에 숨겼는지 사용자가 맞추는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
  ___    ___    ___  
 |   |  |   |  |   | 
 | 1 |  | 2 |  | 3 | 
   O 
 찾았다!
 1, 2, 3 중에서 구슬[O]을 숨긴 컵을 찾으세요 :1


  ___    ___    ___  
 |   |  |   |  |   | 
 | 1 |  | 2 |  | 3 | 
                 O 
        놓쳤다!
 1, 2, 3 중에서 구슬[O]을 숨긴 컵을 찾으세요 :2


  ___    ___    ___  
 |   |  |   |  |   | 
 | 1 |  | 2 |  | 3 | 
          O

알고리즘

프로그램시작 
   3개 컵 중에서 하나에 구슬 숨기기 
   사용자에게 구슬을 숨긴 컵 번호 입력 받기 
   구슬 찾기 결과 출력 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter04/ThreeCupGame.c

	#include <stdio.h>

	#include <stdlib.h>

	#include <time.h>

	 

	// 프로그램시작  

1	int main() {

2		int ball=0, cup=0;

		 

		// 3개 컵 중에서 하나에 구슬 숨기기 

3		srand( time( NULL ) );

4		ball = rand() % 3 + 1;

5		printf("  ___    ___    ___  \n");

6		printf(" |   |  |   |  |   | \n");

7		printf(" | 1 |  | 2 |  | 3 | \n\n\n\n");

	 

		// 사용자에게 구슬을 숨긴 컵 번호 입력 받기  

8		printf(" 1, 2, 3 중에서 구슬[O]을 숨긴 컵을 찾으세요 : ");

9		scanf( "%d", &cup );

			 

		// 구슬 찾기 결과 출력 

10		printf(" \033[4;%dfO\n \033[5;%df", ball*7-3, cup*7-5 );

11		( cup == ball ) ?

12			printf("찾았다!\n")

13		:

14			printf("놓쳤다!\n");

	 

		// 프로그램 종료 

15		return 0;

	}
