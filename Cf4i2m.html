문제

동전을 임의로 던지면 어느 면이 위쪽을 향하도록 떨어졌는지를 사용자가 맞추는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
동전 던지기 게임을 시작합니다.
앞면(0), 뒤면(1) 중 하나를 선택하세요 : 0
동전 앞면을 선택했습니다.
결과는 동전 뒤면입니다. 
틀렸습니다.

프로그램 코드

	#include <stdio.h>
	#include <stdlib.h>							
	#include <time.h>							
	
1	int main() {
2		int headsOrTails = 0, user = 0;
3		srand( time( NULL ) );						
4		headsOrTails = rand() % 2;					
	
5		printf("동전 던지기 게임을 시작합니다.\n");
6		printf("앞면(0), 뒤면(1) 중 하나를 선택하세요 : ");
7		scanf( "%d", &user );
		
8		( user == 0 ) ?					
9			printf( "동전 앞면을 선택했습니다." ) 
10		:								
11			printf( "동전 뒤면을 선택했습니다.");
	
12		( headsOrTails == 0 ) ?				
13			printf( "결과는 동전 앞면입니다. " )
14		:								
15			printf("결과는 동전 뒤면입니다. ");
	
16		( headsOrTails == user ) ?				
17			printf( "맞췄습니다.\n" ) 
18		:								
19			printf("틀렸습니다.\n");
	
20		return 0;
	}
