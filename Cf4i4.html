문제

사용자가 차를 주문하면 따뜻한 차를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
녹차(1), 레몬차(2), 홍차(3) 중 하나를 숫자로 입력하세요: 2

녹차(1), 레몬차(2), 홍차(3) 중 하나를 숫자로 입력하세요: 1

 |____|


녹차(1), 레몬차(2), 홍차(3) 중 하나를 숫자로 입력하세요: 3

 |         |/
 |____|

알고리즘

프로그램 시작 
   사용자가 차를 주문 
   빈컵에 
   주문받은 차의 티백을 넣고 
      1은 녹차 
      2는 레몬차 
      나머지는 홍차 
   물 붓기 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter04/HotTea.c 

	#include <stdio.h> 

		 

	// 프로그램 시작 

1	int main() { 

2		int hotTea = 0; 

	 

		// 사용자가 차를 주문 

3		printf( "녹차(1), 레몬차(2), 홍차(3) 중 하나를 숫자로 입력하세요: "); 

4		scanf( "%d", &hotTea ); 

 

		// 빈컵에  

5		printf("\n |    |\\"); 

6		printf("\n |    |/"); 

7		printf("\n |____|"); 

		 

		// 주문받은 차의 티백을 넣고  

8		switch ( hotTea ) { 

			// 1은 녹차 

9			case 1:					 

10				printf(" \033[42m"); 

11				break;								 

			// 2는 레몬차 

12			case 2:					 

13				printf(" \033[43m"); 

14				break;								 

			// 나머지는 홍차 

15			default:					 

16				printf(" \033[41m"); 

		} 

		 

		// 물 붓기 

17		printf(" \033[3;3f    "); 

18		printf(" \033[4;3f    "); 

19		printf(" \033[5;3f____ \033[0m"); 

	 

		// 프로그램 종료 

20		return 0; 

	}
