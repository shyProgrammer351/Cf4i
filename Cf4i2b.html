문제

증가연산자(++)와 감소연산자(--)의 전위연산자와 후위연산자를 실습하고 그 결과를 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
pre  = ++2 는 2+1( 3 ) 실행후 pre = 3
post = 2++ 는 post = 2 실행후 2+1( 3 )
pre  = --2 는 2-1( 1 ) 실행후 pre = 1
post = 2-- 는 post = 2 실행후 2-1( 1 )

코드 빈칸 채우기

#include <stdio.h> 
 
 main() { 
	 two  2, pre  0, post  0; 
 
	pre   
	      ;									 
	printf( "pre   ++2 는 2+1( 3 ) 실행후 pre  %d\n", pre ); 		 
 
	two  2; 
	post    
	       ;									 
	printf( "post  2++ 는 post  %d 실행후 2+1( 3 )\n", post ); 		 
 
	two  2; 
	pre   
	      ;									 
	printf( "pre   --2 는 2-1( 1 ) 실행후 pre  %d\n", pre ); 		 
 
	two  2; 
	post   
	       ;									 
	printf( "post  2-- 는 post  %d 실행후 2-1( 1 )\n", post ); 		 
 
	return 0; 
}
