문제

증가연산자(++)와 감소연산자(--)의 전위연산자와 후위연산자를 실습하고 그 결과를 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
pre  = ++2 는 2+1( 3 ) 실행후 pre = 3
post = 2++ 는 post = 2 실행후 2+1( 3 )
pre  = --2 는 2-1( 1 ) 실행후 pre = 1
post = 2-- 는 post = 2 실행후 2-1( 1 )

알고리즘

프로그램 시작 
   증가연산자 ++의 전위연산자와 후위연산자 실습 및 출력 
   감소연산자 --의 전위연산자와 후위연산자 실습 및 출력 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter04/IncrementDecrement.c

	#include <stdio.h>

	 

	// 프로그램 시작 

1	int main() {

2		int two = 2, pre = 0, post = 0;

	 

		// 증가연산자 ++의 전위연산자와 후위연산자 실습 및 출력   

3		pre =

4		      ++two;

5		printf( "pre  = ++2 는 2+1( 3 ) 실행후 pre = %d\n", pre );

	 

6		two = 2;

7		post =

8		       two++;

9		printf( "post = 2++ 는 post = %d 실행후 2+1( 3 )\n", post );

	 

		// 감소연산자 --의 전위연산자와 후위연산자 실습 및 출력 

10		two = 2;

11		pre =

12		      --two;

13		printf( "pre  = --2 는 2-1( 1 ) 실행후 pre = %d\n", pre );

	 

14		two = 2;

15		post =

16		       two--;

17		printf( "post = 2-- 는 post = %d 실행후 2-1( 1 )\n", post );

	 

		// 프로그램 종료 

18		return 0;

	}
