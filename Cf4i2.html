문제

증가연산자(++)와 감소연산자(--)의 전위연산자와 후위연산자를 실습하고 그 결과를 출력하는 문제입니다 이를 해결하는 프로그램을 작성하세요 
pre  = ++2 는 2+1(3) 실행후 pre에 3 대입
post = 2++ 는 post에 2 대입후 2+1(3) 실행
pre  = --2 는 2-1(1) 실행후 pre에 1 대입
post = 2-- 는 post에 2 대입후 2-1(1) 실행

알고리즘 설계

프로그램 시작 
   증가연산자 ++의 전위연산자와 후위연산자 실습 및 출력 
   감소연산자 --의 전위연산자와 후위연산자 실습 및 출력 
프로그램 종료

프로그램 구현

     
  
 ※ 단계별 점검: A키(이전), D키(다음) 
   
	// 파일명 : ./Chapter04/IncrementDecrement.c

	#include <stdio.h>

		 

	// 프로그램 시작 

1	int main() {

		// 증가연산자 ++의 전위연산자와 후위연산자 실습 및 출력   

2		int two = 2;

3		int pre =

4		          ++ two;

5		printf( "pre  = ++2 는 2+1(3) 실행후 pre에 %d 대입\n", pre );

		 

6		two = 2;

7		int post =

8		           two ++;

9		printf( "post = 2++ 는 post에 %d 대입후 2+1(3) 실행\n", post );

		 

		// 감소연산자 --의 전위연산자와 후위연산자 실습 및 출력 

10		two = 2;

11		pre =

12		      -- two;

13		printf( "pre  = --2 는 2-1(1) 실행후 pre에 %d 대입\n", pre );

		 

14		two = 2;

15		post =

16		       two --;

17		printf( "post = 2-- 는 post에 %d 대입후 2-1(1) 실행\n", post );

		 

		// 프로그램 종료 

18		return 0;

	}
