문제

사용자에게 체험학습 시작일과 종료일을 입력받아 그 결과를 출력하는 문제입니다 실행순서를 클릭하세요 
이달 1일부터 31일 중에서 체험학습 시작일을 입력하세요: 15
이달 1일부터 31일 중에서 체험학습 종료일을 입력하세요: 18
당신은 체험학습을 15일부터 18일까지 3박4일 다녀오는군요
뜻 깊고 즐거운 시간 보내시길 바래요

프로그램 코드

	#include <stdio.h>


1	int main() {

2		printf( "이달 1일부터 31일 중에서 체험학습 시작일을 입력하세요: " );

		int startDate = 0;

3		scanf( "%d", &startDate );


4		printf( "이달 1일부터 31일 중에서 체험학습 종료일을 입력하세요: " );

		int endDate = 0;

5		scanf( "%d", &endDate );


6		int nights = endDate - startDate;

7		int days = endDate - startDate + 1;


8		printf( "당신은 체험학습을 %d일부터 %d일까지 %d박%d일 다녀오는군요\n", startDate, endDate, nights, days );

9		printf( "뜻 깊고 즐거운 시간 보내시길 바래요 \n" );


10		return 0;

	}
